## Prerequisites

- Node.js 18+
- npm/yarn/pnpm

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sud-seid/craftedbysuda.git
   cd craftedbysuda
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

## Development

```bash
# Start development server
npm run dev

# Start with Turbopack (experimental)
npm run dev --turbo

# Build for production
npm run build

# Start production server
npm run start

# Lint code
npm run lint
```
